## Smart questions to ask before getting tattooed

Many clients only send a reference and ask for a price, but better questions include: will this design fit my placement, what size works best, should the layout be adjusted, and will the details still look clear after healing?

### An unusual but useful question

Ask the artist: “If this tattoo becomes 30% smaller, which details should be removed?” This helps avoid designs that are too crowded for the final size.

## How to message for better tattoo advice

When contacting Lan Pink, describe your idea clearly instead of sending only one reference image. The artist can advise better when they know the placement, approximate size, style, and budget range.

### Example message

You can write: “I want a forearm tattoo, around 8cm, fine line style, with a budget around..., can you advise if this reference works?”

## When should you send a real placement photo?

If you want a cover up, tattoo restoration, or a tattoo on a curved area such as shoulder, ribs, or wrist, a real placement photo helps the artist plan the layout more accurately.
